# COSC3P94_Airline_Website

SkyRoute features a clean, modern airline booking interface designed to make navigation simple and intuitive. The layout uses a consistent card-based design, clear typography, and structured spacing to help users easily find flights, view details, and progress through the booking process.

The interface emphasizes clarity and usability, with organized sections for search results, flight information, and user input. Each page follows a cohesive design system to ensure a smooth and seamless experience across the entire website.
